# molemole

## Overview

`MoleMole` like game


## Charactors

- 主人公 @
- はしご >
- 壁 -|+
- 土 .
- 石 #
- 芋 *
- ドア D
- 何もない （半角スペース）

- 地図には主人公は含めない
- 地図を描いた後に主人公を重ね書きする
