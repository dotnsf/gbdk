# molemole

## Overview

`MoleMole` like game


## Charactors

- 0. 何もない （半角スペース）
- 1. 主人公 @
- 2. 土 .
- 3. はしご >
- 4. 芋 *
- 5. 石 #
- 6. ドア D


## Algorhythm

- 地図は２次元配列
- 地図には主人公は含めない
- 地図を描いた後に主人公を重ね書きする
